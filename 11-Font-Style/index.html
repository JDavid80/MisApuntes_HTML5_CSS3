Font-Style

Italica

Para utilizar itálica, se puede directamente desde el archivo
.html utilizando la etiqueta <i></i> o también podemos
utilizar la etiqueta <em></em>. La diferencia entre los
dos es que la etiqueta em es reconocida por los lectores de pantalla.
También se puede convertir texto en itálica desde el archivo .css
utilizando la sintaxis font-style: italic;

Negrita

Al igual que la itálica, se puede directamente desde el archivo
.html utilizando la etiqueta <b></b> o también podemos
utilizar la etiqueta <strong></strong>. La diferencia entre los
dos es que la etiqueta strong es reconocida por los lectores de pantalla.
También se puede convertir texto en negrita desde el archivo .css
utilizando la sintaxis font-weight: 900; o font-weight: bolder;
