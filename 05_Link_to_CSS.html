Enlazar Hojas de Estilo desde HTML

Linking your css to your html

El código para enlazar una hoja de estilo va entre las etiquetas head

Podriamos dividir el código para enlazar con una hoja de estilo en 3 "ecuaciones":

1er. Ecuación: link rel="stylesheet". En este caso, la palabra rel viene de relationship, que significa que el link esta relacionado con una hoja de estilo

2da. Ecuación: type="text/css" Nos indica que el tipo de texto es css.

3er. Ecuación: href="css/styles.css". href significa hypertext reference.
